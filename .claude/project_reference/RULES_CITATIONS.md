## Key Corrections Discovered

These are rule interpretations we had wrong initially and caught during verification:

**Aetregan Wis 11 vs 12** — Attribute boosts are +2 to score or +1 to modifier, not +1 to score. Therefore starting from 10, a single boost yields 12 (mod +1), not 11. Caught in CP0.5.

**Mortar EV 5.95 vs 5.60** — Brief originally undercounted critical failures on mortar saves. The rule says "total ≤ DC-10 is crit failure," and ≤ includes equality. So for DC 17 vs +5 Reflex, rolls 1 and 2 (totals 6 and 7) both crit-fail. Correct EV is 5.95, not 5.60. Caught in CP0.5.

**Banner aura 30 vs 40 when planted** — Base Commander banner is a 30-ft emanation. Plant Banner feat explicitly expands to a 40-ft burst. Originally hardcoded as 30 regardless. Caught in CP4.

**Scorpion Whip vs Whip** — Aetregan actually uses Scorpion Whip (lethal), not Whip (nonlethal). Same core stats except the nonlethal trait is absent. Caught in CP4.5.

**Perception trained vs expert** — Commander gets expert Perception at level 1, not trained. Aetregan's Perception is +6 (Wis +1 + expert +5), not +4. Caught in CP4.5.

## Citation Format in Code

In docstrings:

```python
def foo(...):
    """Compute X.
    
    (AoN: https://2e.aonprd.com/Rules.aspx?ID=2187)
    """
```

Multiple citations on separate lines:

```python
def complex_function(...):
    """Compute Y affected by multiple rules.
    
    (AoN: https://2e.aonprd.com/Rules.aspx?ID=2187)
    (AoN: https://2e.aonprd.com/Conditions.aspx?ID=58)
    (AoN: https://2e.aonprd.com/Traits.aspx?ID=684)
    """
```

For inline comments explaining a specific rule application:

```python
# Plant Banner expands aura to 40-ft burst
# (AoN: https://2e.aonprd.com/Feats.aspx?ID=7796)
radius = 40 if self._banner_planted else 30
```

Every non-obvious rule application should have a citation.
